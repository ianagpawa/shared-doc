# Shared Doc
A simple collaborative document editing app built with React, Node and websockets.

![](./media/preview.gif)

## How to run the app?

```bash
#-- Setup and start the server
cd server
npm install # or yarn install
npm start # or yarn start

#-- Setup and start the client
cd client
npm install # or yarn install
npm start # or yarn start
```
